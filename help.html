SUSTech 自动排课 · 帮助文档

项目简介

本项目旨在帮助同学快速生成无时间冲突的课程表。你可以搜索并选择课程，设置优先级，系统会自动组合出多份可行方案，并支持导出图片与 CSV。

主要功能

  • 课程搜索：按课程名称或教师名搜索。
  • 已选课程管理：
    • 拖拽调整优先级（靠前优先保留）。
    • 开关单门课程的启用/禁用，不想参与本次排课就关闭开关。
  • 自动排课：
    • 将主课与其附属课（如实验/讨论）自动成组。
    • 生成不冲突的多份方案；若“全选课程组”无解，采用“最少跳课”策略，从跳 1 门开始寻找可行解，尽量保留高优先级课程。
    • 最大方案数默认限制为 256 份。
  • 课表渲染：
    • 单双周分表展示；支持周六、周日显示（兼容周日编码为 0 或 7 的数据）。
    • 分页显示当前方案编号与“课程数”。
    • 点击任意课程格子，会为该课程的所有格子随机更换背景色。
  • 导出：
    • 导出课表为图片（PNG）。
    • 导出当前方案为 CSV（包含单双周矩阵与课程详情）。
  • 本地存储：自动保存已选课程与开关、优先级设置到浏览器本地。

使用指南

  1. 进入首页，点击“开始排课”。
  2. 在排课页搜索框中输入课程或教师，加入到“已选课程”。
  3. 在“已选课程”中：
    • 拖拽条目可调整优先级（靠前更优先保留）。
    • 使用开关可临时禁用某门课程参与排课。
  4. 点击“生成课表”或按 Enter 进入课表页。
  5. 在课表页：
    • 使用左右分页按钮或快捷键浏览不同方案。
    • 查看“课程数”，了解当前方案包含的课程数量（主课+附属课算作一门）。
    • 点击课程格子可随机更换该课程的配色（同一课程的所有格子会同步变色）。
    • 可导出图片或 CSV 保存。

快捷键

  • 在搜索页：I 聚焦搜索框，Enter 生成课表。
  • 在课表页：
    • ← 或 H：上一份方案
    • → 或 L：下一份方案
    • Shift + ←/H 或 →/L：按约 10% 份数快速跳页
    • Backspace：返回选课页面

导出说明

  • 导出图片：截取当前课表区域，文件名包含方案编号。
  • 导出 CSV：包含单/双周两张 5x7 矩阵及课程详情（课程名、教师、课程ID、院系、时间安排）。

注意事项

  • 最少跳课策略：当所有课程组一起排无法得到无冲突方案时，系统会从“跳 1 门”开始寻找可行解。若希望某门课优先保留，请将其拖到更靠前的位置。
  • 附属课识别：仅当某课程 ID 末位为字母且存在同前缀的主课时，才会被识别为附属课并与主课成组。
  • 周末显示：若周末未正确显示，请提供该课程时间编码（是否用 0 表示周日）以便进一步兼容。
  • 随机配色：每次渲染或翻页会重新分配基础颜色；点击格子只在当前页面会话中生效，不做长期保存。
  • 浏览器建议：建议使用最新版 Chrome/Edge。
  • 本地数据清理：若页面异常，可清理站点存储，或在控制台执行 localStorage.removeItem('selectedCourses') 后刷新。
